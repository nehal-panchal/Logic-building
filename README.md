# Logic-building

Level 1: Simple Conditions (Getting Started)

1. Take a number and print whether it is positive, negative, or zero.

2. Check if a number is even or odd.

3. Check if a number is divisible by 5.

4. Check if a number is divisible by both 3 and 5.

5. Check if a given year is a leap year.

6. Take two numbers and print the larger one.

7. Take three numbers and print the largest number.

8. Take a temperature value and print "Cold", "Warm", or "Hot" using range conditions.

9. Take a character and check if it is a vowel or consonant.

10. Take a character and check whether it is: Uppercase letter , Lowercase letter ,Digit Special character
