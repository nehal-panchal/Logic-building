# Logic-building

# Level 1: Simple Conditions (Getting Started)

1. Take a number and print whether it is positive, negative, or zero.
2. Check if a number is even or odd.
3. Check if a number is divisible by 5.
4. Check if a number is divisible by both 3 and 5.
5. Check if a given year is a leap year.
6. Take two numbers and print the larger one.
7. Take three numbers and print the largest number.
8. Take a temperature value and print "Cold", "Warm", or "Hot" using range conditions.
9. Take a character and check if it is a vowel or consonant.
10. Take a character and check whether it is: Uppercase letter , Lowercase letter ,Digit Special character

# Level 2: Nested If & Multiple Conditions

11. Take three sides and check if they form a valid triangle.
12. If the sides form a valid triangle, determine whether it is equilateral, isosceles, or scalene.
13. Take marks (0–100) and print the corresponding grade (A / B / C / D / F).
14. Check if one of two given numbers is a multiple of the other.
15. Take the hour of the day (0–23) and print:
    “Good Morning”
    “Good Afternoon”
    “Good Evening”
    “Good Night”
16. Check voting eligibility for a given age (18+).
17. Take two numbers and determine whether:
    both are even
    both are odd
    one is even and one is odd

18. Take an alphabet character and check if it lies between:
    ‘a’ and ‘m’, or
    ‘n’ and ‘z’
19. Take a day number (1–7) and print the corresponding day name.
20. Take a month number (1–12) and print the number of days in that month (ignore leap years).
